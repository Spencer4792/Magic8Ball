# Magic 8-Ball

An interactive Magic 8-Ball built with HTML, CSS, and JavaScript. Click the ball to shake it and reveal your fortune!

## Features

- Realistic 3D ball with glossy highlights and shading
- Smooth shake animation with natural physics-based decay
- Classic blue triangle window that displays answers
- 9 traditional Magic 8-Ball responses
- Starry animated background
- Fully responsive design

## Demo

Simply open `magic8ball.html` in any modern web browser to play.

## How to Use

1. Click on the 8-ball
2. Watch it shake
3. Read your fortune
4. Click again for a new prediction

## Responses

The Magic 8-Ball includes all 9 classic responses:

- It is certain
- It is decidedly so
- Yes
- Reply hazy try again
- Ask again later
- Concentrate and ask again
- My reply is no
- Outlook not so good
- Very doubtful

## Technologies Used

- HTML5
- CSS3 (animations, gradients, flexbox)
- Vanilla JavaScript
